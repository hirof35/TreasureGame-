THE TREASURE HUNTER - Sound Edition
PythonとTkinterで構築された、音と直感で挑むレトロスタイルの迷宮探索ゲームです。
限られた時間内に、呪われた迷宮に隠された秘宝を見つけ出してください。

🌟 特徴
シャッフル迷宮システム: 罠（TRAP）を踏むと、宝や地図の場所がすべて再配置されます。一度通った道も安全とは限りません。

音による演出: pygame.mixer を使用したサウンドエンジンを搭載。移動音や勝利時のファンファーレが探索を盛り上げます。

戦略的なヒント:

画面下部に「宝までのマンハッタン距離」を表示。

罠が隣接していると「嫌な予感がする...」という警告が発生。

古びた地図: 迷宮内に落ちている地図（M）を入手すれば、すべての罠の場所が明らかになります。

低速移動・精密操作: キーボードを使わず、マウスによるボタンクリックで一歩ずつ慎重に進む探索スタイル。

🕹 ルール
制限時間: 30秒

移動: 現在地（YOU）から上下左右に隣接するマスをクリック。

ペナルティ: 罠を踏むと残り時間が -5秒 され、スタート地点に強制送還。さらに、宝を含むすべての配置が書き換えられます。

クリア条件: 制限時間内に宝（隠しマス）に到達すること。

🛠 技術スタック
Core: Python 3.x

GUI: Tkinter (Standard Library)

Audio: Pygame (Mixer module)

Algorithm:

ランダム配置アルゴリズム（重複回避ロジック）

マンハッタン距離計算による距離フィードバック

🚀 インストール & 実行
pygame ライブラリが必要です。

Bash
# クローン
git clone https://github.com/your-username/treasure-hunter.git

# 依存ライブラリのインストール
pip install pygame

# ゲーム開始
python main.py
Note: 音源ファイル（bgm.mp3, win.wav 等）を配置することで、フルサウンドでプレイ可能です（現在はファイルがない場合でも警告音で代用される設計になっています）。
